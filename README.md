Weekly Test 3 - HTML & CSS
This repository contains a simple HTML and CSS project for a web page layout. 
The web page includes a header, navigation bar, main content section, and images.
The navigation bar is sticky and hides when scrolling down, revealing the main content, and reappears when scrolling up.


HTML Structure
The index.html file contains the following structure:

A header with a message about free shipping.
A sticky navigation bar with various navigation links, including a search input field.
A main content section with information about a sale, including an image and text.
A middle content section with trending products.
An end content section with a text description and an image of Air Jordan's first shoe.

CSS Styles
The styles.css file contains CSS styles for the web page. Key styles include:

Styling for the header and navigation bar.
Styling for the main content, including an image and text.
Styling for the middle content, displaying trending products.
Styling for the end content, including text and an image with a hover effect.
